Status

This repository is a research handoff package, not a claimed proof of the Riemann Hypothesis.

It contains a conditional dyadic Nyman–Beurling framework, exact finite-dimensional identities, reproducible computations, and a proposed missing theorem called the block no-stalling criterion. If that missing theorem were proved, it would imply the Riemann Hypothesis through the Nyman–Beurling criterion.

The purpose of this repository is to make the ideas, computations, code, and provenance public so that mathematicians and computational number theorists can verify, critique, extend, or disprove the approach.
